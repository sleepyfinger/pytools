# YouTube Downloader (yt-dlp 기반)

이 프로그램은 파이썬과 `yt-dlp` 라이브러리를 사용하여 유튜브 영상(MP4) 또는 오디오(MP3)를 고화질/고음질로 다운로드할 수 있는 도구입니다.

## 기능

- **비디오 다운로드**: 유튜브에서 제공하는 최고 화질의 영상과 오디오 트랙을 병합하여 MP4 파일로 저장합니다.
- **오디오 다운로드**: 영상에서 오디오만 추출하여 MP3 파일로 변환 및 저장합니다.

## 사전 준비 사항 (Prerequisites)

이 프로그램을 정상적으로 실행하기 위해서는 다음 항목들이 시스템에 설치되어 있어야 합니다.

1.  **Python 3.x**: 파이썬 실행 환경.
2.  **FFmpeg (필수)**: 비디오와 오디오를 합치거나(Video 모드), 오디오를 변환(Audio 모드)하는 데 반드시 필요합니다.
    - **Windows**: FFmpeg 공식 사이트에서 빌드를 다운로드한 후, `ffmpeg.exe` 파일을 `main.py`와 같은 폴더에 두거나 시스템 환경 변수(PATH)에 추가해야 합니다.
    - **Mac**: 터미널에서 `brew install ffmpeg` 실행.
    - **Linux**: 터미널에서 `sudo apt install ffmpeg` 실행.

## 설치 방법

터미널(CMD)에서 아래 명령어를 입력하여 필수 라이브러리를 설치하세요.

```bash
pip install yt-dlp
```

## 사용 방법

1.  터미널에서 프로그램을 실행합니다.
    ```bash
    python main.py
    ```
2.  프롬프트에 유튜브 URL을 입력합니다. (종료하려면 `q` 입력)
3.  원하는 다운로드 형식을 선택합니다.
    - `1`: 비디오 (영상 + 소리)
    - `2`: 오디오 (MP3 소리만)
4.  다운로드가 완료되면 프로그램이 위치한 폴더에 파일이 저장됩니다.

## 주의사항

다운로드한 콘텐츠는 **개인 소장용**으로만 사용해야 하며, 타인에게 공유하거나 상업적으로 이용할 경우 저작권법에 위배될 수 있습니다.
